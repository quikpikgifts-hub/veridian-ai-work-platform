# Veridian Risk Platform

Production-ready Next.js operational risk management platform.

## Stack
- Next.js 16 + TypeScript
- Tailwind CSS v4
- Recharts (data visualization)
- Lucide React (icons)

## Pages
- `/` — Landing page
- `/login` — Authentication
- `/dashboard` — Operations command center
- `/incidents` — Incident registry with filtering
- `/threat-intel` — Intelligence feed
- `/settings` — Platform configuration

## Deploy to Vercel
```bash
npm install
npm run build
vercel --prod
```

## Local Development
```bash
npm install
npm run dev
# Open http://localhost:3000
```
